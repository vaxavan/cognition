# 🎨 Frontend для тестов RAG-архитектуры

Простой интерфейс для загрузки и обработки файлов.

## 🚀 Быстрый старт

1. Убедись, что бэкенды запущены:
   ```bash
   # В одном окне:
   cd chat-service && uvicorn app.main:app --port 8000

   # В другом окне:
   cd context-engine && uvicorn context_engine.main:app --port 8002
   ```

2. Открой `frontend/index.html` в браузере:
   - Просто перетащи файл в браузер, ИЛИ
   - Запусти локальный сервер (рекомендуется для CORS):
     ```bash
     cd frontend
     python -m http.server 8080
     ```
   - Открой: `http://localhost:8080`

## 📋 Как работает

1. Выбираешь файл + проект → нажимаешь "Запросить загрузку"
2. Получаешь presigned URL → нажимаешь "Загрузить в MinIO"
3. Файл грузится напрямую в MinIO (без прокси)
4. Нажимаешь "Запустить обработку" → вызывается context-engine
5. Видишь чанки и результат!

## 🔧 Если CORS блокирует загрузку в MinIO

Для локальных тестов интерфейс продолжает работу в "мок-режиме".
Для продакшена настрой CORS в MinIO.

## 🎯 Фичи

- ✅ Чистый HTML+CSS+JS — без сборки, без Node.js
- ✅ Пошаговый интерфейс с визуальной обратной связью
- ✅ Логирование всех запросов в консоль интерфейса
- ✅ Обработка ошибок и понятные сообщения
- ✅ Адаптивный дизайн (работает на мобильных)
- ✅ Мок-режим для тестов без реального MinIO
